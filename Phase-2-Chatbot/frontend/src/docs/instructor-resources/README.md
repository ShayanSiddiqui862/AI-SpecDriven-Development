## Laboratory Setup Guide

### Hardware Requirements
- Workstations with RTX 3080 or better (for Isaac Sim)
- RealSense D435i cameras
- LiDAR sensors (optional)
- IMU units
- Networked humanoid robots (Unitree Go2, G1, or alternatives)

### Software Installation
1. Ubuntu 22.04 LTS installation
2. ROS 2 Humble Hawksbill setup
3. Gazebo Garden installation
4. Isaac Sim 2023.1+ deployment
5. Development environment configuration

### Network Configuration
- ROS network setup for multi-robot systems
- Simulation to real robot communication
- Cloud instance connectivity

## Teaching Tips

### Effective Demonstrations
- Start with simple ROS nodes before complex systems
- Use simulation before real hardware
- Provide clear, step-by-step examples
- Encourage collaborative debugging

### Common Student Challenges
- Understanding ROS 2 architecture concepts
- Debugging multi-package systems
- Managing complex simulation environments
- Integrating AI components with robotics

### Assessment Strategies
- Focus on system integration rather than isolated components
- Emphasize iterative development and testing
- Encourage documentation and code quality
- Provide regular feedback on project milestones

## Accessibility Resources

### Accommodation Guidelines
- Alternative interfaces for students with mobility limitations
