# System Architecture Diagrams

This directory contains system architecture diagrams for the Physical AI & Humanoid Robotics textbook.

## Diagrams Included:

1. **Robotic Nervous System Architecture** - `robotic-nervous-system.svg`
   - Shows the ROS 2 architecture with nodes, topics, and services
   - Illustrates the communication between different components

2. **Digital Twin Architecture** - `digital-twin-architecture.svg`
   - Shows the integration between Gazebo, Unity, and ROS 2
   - Illustrates data flow between simulation and control systems

3. **AI-Robot Brain Architecture** - `ai-robot-brain-architecture.svg`
   - Shows the integration of Isaac Sim, perception stack, and navigation
   - Illustrates the VSLAM and Nav2 components

4. **VLA System Integration** - `vla-system-integration.svg`
   - Shows the integration of voice processing, LLM, and action execution
   - Illustrates the complete pipeline from natural language to robot action

5. **Hardware Connection Schematics** - `hardware-connection-schematics.svg`
   - Shows how different sensors connect to the robot's main computer
   - Includes RealSense, LiDAR, IMU, and other sensor connections

6. **Data Flow Architecture** - `data-flow-architecture.svg`
   - Shows the complete data flow from sensors to actuators
   - Illustrates how different systems interact
